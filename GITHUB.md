# GitHub profile kit

Conținut gata de copiat pe GitHub. Înlocuiește `[Your Name]` și `YOUR-USERNAME` peste tot.

---

## 1. Bio scurt (câmpul „Bio" din profil — limită 160 caractere)

Varianta recomandată (148 caractere):

```
PhD student in Industrial Engineering | Injection molding of technical polymers · Additive manufacturing · 3D metrology (CMM) | Open science
```

Variantă alternativă, mai personală (124 caractere):

```
PhD researcher — injection molding, additive manufacturing, dimensional metrology. I measure what we make, and make it better.
```

---

## 2. Profile README (apare pe pagina ta de profil)

Cum se activează: creează un repository nou cu numele EXACT egal cu username-ul tău
(ex. repo `YOUR-USERNAME` pentru contul `YOUR-USERNAME`), public, bifează „Add a README file",
apoi lipește conținutul de mai jos în acel README.md.

```markdown
# [Your Name]

**PhD student in Industrial Engineering.** I research how injection molding of
technical polymers and additive manufacturing (MEX/FDM) can be optimized through
dimensional metrology — coordinate measuring machines (CMM), 3D scanning and
design of experiments (DoE).

## Research areas

- **Injection molding of technical polymers** — process parameters, semi-crystalline
  polymers and the dimensional quality of molded parts
- **Additive manufacturing (MEX/FDM)** — metrology-driven dimensional optimization
- **3D dimensional metrology** — CMM, GD&T evaluation, scan-vs-CAD inspection
- **Industry 4.0 & digital twins** — data-driven process monitoring

## What you'll find here

- The source of my research website — publications, conference posters and a
  [Knowledge Hub](https://YOUR-USERNAME.github.io/knowledge.html) on polymers
  (from Goodyear's 1844 vulcanization patent to PEEK and FDM)
- Python tools for CMM data analysis, process capability and DoE *(in development)*

## Find me

[Website](https://YOUR-USERNAME.github.io) ·
[ORCID](https://orcid.org/0000-0000-0000-0000) ·
[Google Scholar](#) ·
[ResearchGate](#) ·
[LinkedIn](#)

> Open to collaboration on polymer processing, additive manufacturing and
> dimensional metrology research.
```

## 3. Câmpurile de profil (Settings → Public profile)

| Câmp | Ce pui |
|---|---|
| Name | Numele tău (identitate hibridă: nume + atât) |
| Bio | textul de la punctul 1 |
| Company | Universitatea ta (ex. `@universitate` dacă are organizație GitHub) |
| Location | Orașul, Țara (opțional) |
| Website | `https://YOUR-USERNAME.github.io` |
| Social accounts | ORCID, LinkedIn, Scholar (max 4 linkuri) |

Sfat: activează 2FA (Settings → Password and authentication) înainte de orice.
